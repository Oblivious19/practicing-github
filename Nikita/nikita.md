## Basic Git Hub commands


### Steps to clone repo:

 1. Copy the repository
 2. Open Git bash
 3. In git bash enter
    ```bash
    cd desktop
    ```
    the above cmd is to copy repo on the desktop
 4. To clone repo
     ```bash
    git clone (link)
    ```
### Basic Commands:

 1.
    ```bash
    git add .
    ```
 2. 
     ```bash
    git commit -m "[ADD]"
    ```
 3. 
     ```bash
    git status
    ```
 4. 
    ```bash
    git pull origin main
    ```
 5. 
    ```bash
    git switch 3-basic (issue no. & file name)
    ```
 6. 
    ```bash
    git push
    ```
