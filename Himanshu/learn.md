## Steps to write Content in Markdown

 1. Save the file with `.md` extension
 2. Header can be defined by using `#` 
    Example: 
    # About
    ## About
    ### About

 3. To write a program highlight it using ``
    Example:
    ```bash
    git commit -m
    ```
